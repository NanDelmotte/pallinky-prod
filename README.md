## Getting started

1. Install dependencies

   ```bash
   npm install
   ```

2. Create `.env.local` based on `.env.example`

3. Run dev server

   ```bash
   npm run dev
   ```

4. Open:

   - http://localhost:3000
   - http://localhost:3000/login
